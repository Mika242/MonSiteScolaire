a. Le cercle trigonométrique

Enroulement de la droite réelle · Angles remarquables · Parité et périodicité

🖱️ Déplace ta souris sur le cercle pour lire les valeurs de sin et cos en temps réel

Angle0°
sin(x)0.000

📖 Définition — Le cercle trigonométrique

Le cercle trigonométrique est le cercle de centre O et de rayon 1 dans un repère orthonormé. On enroule la droite réelle autour de ce cercle : à tout réel x on associe un point M(x) sur ce cercle.

M est le point du cercle unité tel que arc(I, M) = x (sens trigonométrique = positif)
💡 Sens trigonométrique Sens antihoraire (sens inverse des aiguilles d'une montre). On part de I = (1, 0).

Q7 Avancé

Sans calculatrice, donne les valeurs exactes de sin(−π/3) et cos(7π/4).

sin(−π/3) : sin est impaire → sin(−x) = −sin(x) → sin(−π/3) = −sin(π/3) = −√3/2
cos(7π/4) : 7π/4 = 2π − π/4. Par périodicité : cos(2π − x) = cos(x)
cos(7π/4) = cos(−π/4) = cos(π/4) = √2/2 (cos est paire)
Q8 Avancé

Démontre que sin²x + cos²x = 1 à partir de la définition sur le cercle unité.

M(x) est un point sur le cercle de centre O et de rayon 1
Ses coordonnées sont (cos x ; sin x)
Or OM = 1 donc par Pythagore : OM² = cos²x + sin²x = 1²
Conclusion : sin²x + cos²x = 1 pour tout réel x ✔
Cette relation est appelée la relation fondamentale de la trigonométrie.

b. Sinus et cosinus d'un nombre réel

Propriétés · Parité · Périodicité · Relation fondamentale

📖 Propriétés fondamentales
Relation fondamentale : sin²x + cos²x = 1 Périodicité : sin(x + 2π) = sin(x) · cos(x + 2π) = cos(x) sin paire/impaire : sin(−x) = −sin(x) (impaire) cos paire/impaire : cos(−x) = cos(x) (paire)
Symétriesincos
x → −x (axe des abscisses)sin(−x) = −sin xcos(−x) = cos x
x → π − x (axe des ordonnées)sin(π−x) = sin xcos(π−x) = −cos x
x → π + x (symétrie centrale O)sin(π+x) = −sin xcos(π+x) = −cos x
x → 2π − x (axe abscisses)sin(2π−x) = −sin xcos(2π−x) = cos x
"On utilise sin(π − x) = sin x pour ramener à un angle du 1er quadrant, puis on lit la valeur dans le tableau des valeurs remarquables."

Sachant que cos x = −√3/2 et que x ∈ [π ; 2π], détermine sin x et précise dans quel quadrant se trouve M(x).

cos x = −√3/2 → x est dans les quadrants II ou III (cos négatif)
Mais x ∈ [π ; 2π] → x est dans les quadrants III ou IV
L'intersection : quadrant III → x ∈ ]π ; 3π/2[
sin²x = 1 − cos²x = 1 − 3/4 = 1/4 → sin x = ±1/2
Quadrant III → sin x < 0 → sin x = −1/2
Conclusion : M(x) est dans le quadrant III, x correspond à 5π/6 + π = 7π/6
Q6 Avancé

Calcule la valeur exacte de sin²(π/12) + cos²(π/12), puis de 4sin(π/6)·cos(π/6).

sin²(π/12) + cos²(π/12) = 1 par la relation fondamentale (quelle que soit la valeur)
4sin(π/6)·cos(π/6) = 4 × (1/2) × (√3/2) = 4 × √3/4 = √3
Note : 4sin(x)cos(x) = 2sin(2x) → 2sin(π/3) = 2 × √3/2 = √3 ✔ (formule de duplication)

c. Équations trigonométriques

sin x = a · cos x = a · Méthode sur le cercle

📖 Résoudre sin x = a
Si |a| > 1 : pas de solution Si a ∈ [−1;1] : sin x = a ⟺ x = arcsin(a) + 2kπ ou x = π − arcsin(a) + 2kπ, k ∈ ℤ
Exemple : sin x = 1/2 arcsin(1/2) = π/6
Solution : x = π/6 + 2kπ  ou  x = π − π/6 + 2kπ = 5π/6 + 2kπ, k ∈ ℤ
📖 Résoudre cos x = a
Si |a| > 1 : pas de solution Si a ∈ [−1;1] : cos x = a ⟺ x = ±arccos(a) + 2kπ, k ∈ ℤ
Exemple : cos x = √3/2 arccos(√3/2) = π/6
Solution : x = π/6 + 2kπ  ou  x = −π/6 + 2kπ, k ∈ ℤ
💡 Méthode sur le cercle 1) Tracer le cercle · 2) Placer la valeur sur l'axe correspondant · 3) Trouver les 2 points d'intersection · 4) Lire les angles · 5) Ajouter + 2kπ
"sin x = a admet deux familles de solutions : x = arcsin(a) + 2kπ et x = π − arcsin(a) + 2kπ, k ∈ ℤ."

Q6 Avancé

Résous dans ℝ : 2cos²x − cos x − 1 = 0.

On pose X = cos x. L'équation devient : 2X² − X − 1 = 0
Discriminant : Δ = 1 + 8 = 9 → √Δ = 3
X₁ = (1+3)/4 = 1  ·  X₂ = (1−3)/4 = −1/2
cos x = 1 : x = 0 + 2kπ = 2kπ, k ∈ ℤ
cos x = −1/2 : arccos(−1/2) = 2π/3 → x = ±2π/3 + 2kπ, k ∈ ℤ
Q7 Avancé

Résous sur [0 ; 2π] : sin²x = 3cos²x.

sin²x = 3cos²x ⟺ sin²x/cos²x = 3 ⟺ tan²x = 3 (si cos x ≠ 0)
tan x = ±√3 → x = ±π/3 + kπ, k ∈ ℤ
Sur [0;2π] : x = π/3, x = 2π/3, x = 4π/3, x = 5π/3
Vérification cos ≠ 0 : aucune de ces valeurs n'annule cos ✔

d. Inéquations trigonométriques

sin x ≥ a · cos x ≤ a · Méthode par arc sur le cercle

📖 Méthode générale — Inéquations trig.

Pour résoudre une inéquation, on utilise le cercle trigonométrique et les variations de sin et cos.

InéquationMéthode sur le cercle
sin x ≥ aTracer la droite y = a, lire les arcs où sin est au-dessus
sin x ≤ aTracer la droite y = a, lire les arcs où sin est en-dessous
cos x ≥ aTracer la droite x = a, lire les arcs où cos est à droite
cos x ≤ aTracer la droite x = a, lire les arcs où cos est à gauche
Exemple : sin x ≥ 1/2 sur [0;2π] sin x = 1/2 en x = π/6 et x = 5π/6
sin est au-dessus de 1/2 entre ces deux valeurs.
Solution : x ∈ [π/6 ; 5π/6]
"Pour sin x ≥ a, on résout d'abord sin x = a pour trouver les bornes, puis on détermine l'intervalle où sin dépasse la valeur a grâce aux variations."
📖 Résumé des solutions sur [0 ; 2π]
sin x ≥ 0 ⟺ x ∈ [0 ; π] sin x ≤ 0 ⟺ x ∈ [π ; 2π] cos x ≥ 0 ⟺ x ∈ [0 ; π/2] ∪ [3π/2 ; 2π] cos x ≤ 0 ⟺ x ∈ [π/2 ; 3π/2]

Résous dans ℝ : sin x < −√2/2.

sin x < −√2/2 quand M(x) est dans la partie basse du cercle en-dessous de l'ordonnée −√2/2
Sur [0;2π] : x ∈ ]5π/4 ; 7π/4[
Solution dans ℝ : x ∈ ]5π/4 + 2kπ ; 7π/4 + 2kπ[, k ∈ ℤ
Q6 Avancé

Résous dans ℝ : 2cos²x − 1 ≥ 0.

2cos²x − 1 ≥ 0 ⟺ cos²x ≥ 1/2 ⟺ |cos x| ≥ √2/2
⟺ cos x ≥ √2/2 ou cos x ≤ −√2/2
cos x ≥ √2/2 : x ∈ [−π/4 ; π/4] + 2kπ
cos x ≤ −√2/2 : x ∈ [3π/4 ; 5π/4] + 2kπ
Solution : x ∈ [−π/4 + kπ ; π/4 + kπ], k ∈ ℤ (en notant que la période est π)

📈 Courbes de sin et cos sur [−2π ; 2π]
"La fonction sin est impaire (courbe symétrique par rapport à O) et de période 2π. La fonction cos est paire (courbe symétrique par rapport à l'axe des ordonnées) et de période 2π. cos est décroissante sur [0 ; π] et croissante sur [−π ; 0] (ou [π ; 2π])."

Q6 Avancé

Étude complète de f(x) = 2sin(x − π/4) : période, amplitude, phase, tableau de variations sur [0 ; 2π], maximum et minimum.

Amplitude : A = 2 → f oscille entre −2 et 2
Période : ω = 1 → T = 2π/1 = 2π
Phase : φ = −π/4 → la courbe est décalée de +π/4 vers la droite
Maximum = 2 quand x − π/4 = π/2 → x = 3π/4
Minimum = −2 quand x − π/4 = −π/2 → x = −π/4 (hors [0;2π]) ou x = 3π/2 + π/4 = 7π/4
f(0) = 2sin(−π/4) = −√2 · f(3π/4) = 2 · f(7π/4) = −2 · f(2π) = 2sin(7π/4) = −√2
Q7 Avancé

Résous sur [0 ; 2π] l'inéquation sin x > cos x en étudiant le signe de sin x − cos x.

sin x − cos x = √2 · sin(x − π/4) (formule de linéarisation)
sin x > cos x ⟺ √2·sin(x − π/4) > 0 ⟺ sin(x − π/4) > 0
sin u > 0 ⟺ u ∈ ]0 ; π[ + 2kπ
Donc x − π/4 ∈ ]0 ; π[ ⟺ x ∈ ]π/4 ; π + π/4[ = ]π/4 ; 5π/4[
Solution sur [0;2π] : x ∈ ]π/4 ; 5π/4[
