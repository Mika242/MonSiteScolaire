📘 Définitions essentielles

  • 🔁 Transformation physique : changement d'état sans modification chimique.
  • 💧 Changement d'état : passage d’un état (solide, liquide, gaz) à un autre.
  • 🌡️ Endothermique : le système absorbe de l’énergie (fusion, vaporisation, etc).
  • 🔥 Exothermique : le système libère de l’énergie (solidification, liquéfaction, etc).
  • 📏 Énergie massique de changement d’état : quantité d’énergie absorbée ou cédée par 1 kg de matière pendant le changement d’état.

🧪 Exercices types

🔹 Exercice 1 : Identifier le changement d’état

Un glaçon fond et est placé dans un congélateur. Que se passe-t-il ?

🔹 Exercice 2 : Calculer l’énergie de fusion

Faire fondre 300 g de glace (L = 334×10³ J/kg).

🔹 Exercice 3 : Masse inconnue

Q = 4,2 × 10⁴ J ; L = 210 × 10³ J/kg → Trouver m ?

🔹 Exercice 4 : Identifier les étapes d’un graphique

A → Fusion ; B → Échauffement liquide ; C → Vaporisation.

✅ Corrections des exercices

Exercice 1 : Solidification (liquide → solide), c’est une transformation exothermique car le système libère de l’énergie.
Exercice 2 : Q = 0,3 × 334 000 = 100 200 J — Transformation endothermique.
Exercice 3 : m = Q / L = 4,2×10⁴ / 210×10³ = 0,2 kg
Exercice 4 : A et C sont des transformations endothermiques, car le système absorbe de l’énergie pour changer d’état.

🧠 Astuces pour réussir

  • ✔️ Toujours repérer l’état initial et l’état final.
  • ✔️ Indiquer si l’énergie est reçue ou cédée.
  • ✔️ Utilise la formule Q = m × L pour les calculs.
  • ✔️ Utilise les unités : kg pour la masse, J pour l’énergie.
