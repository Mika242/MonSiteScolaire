Contrôle d'entraînement : L'effort physique et l'appareil cardio-respiratoire

Exercice 1 : Questions à choix multiples

Choisis la ou les bonnes réponses :

  • 1. Pendant un effort physique, le cœur :
    ⬜ bat plus vite
    ⬜ s'arrête
    ⬜ ralentit
  • 2. Le rôle principal des poumons est :
    ⬜ d’éliminer les déchets
    ⬜ d’absorber l’oxygène
    ⬜ de produire du sang
  • 3. Quand on court :
    ⬜ notre respiration s’accélère
    ⬜ le sang circule moins vite
    ⬜ on n’a plus besoin d’oxygène
  • 4. Le dioxygène est transporté par :
    ⬜ les globules blancs
    ⬜ le plasma
    ⬜ les globules rouges
Correction type :
1 : bat plus vite
2 : d’absorber l’oxygène
3 : notre respiration s’accélère
4 : les globules rouges

Exercice 2 : Texte à trous

Complète le texte avec les mots suivants : effort - dioxygène - fréquence - muscles - déchets

Quand on fait un ________ physique, la ________ cardiaque augmente. Le sang transporte plus de ________ vers les ________, qui produisent de l'énergie et des ________.

Correction type :
effort – fréquence – dioxygène – muscles – déchets

Exercice 3 : Questions courtes

  1. Qu’est-ce qui change dans notre respiration pendant l’effort ?
  2. Pourquoi notre cœur bat-il plus vite pendant l’effort ?
  3. Quel est le rôle du sang dans l’effort physique ?
  4. Quelles sont les conséquences d’un manque de dioxygène pour les muscles ?
Correction type :
1. La respiration devient plus rapide et plus profonde.
2. Pour apporter plus de sang et donc d’oxygène aux muscles.
3. Il transporte le dioxygène et les nutriments jusqu’aux muscles.
4. Les muscles peuvent se fatiguer ou produire des crampes.

Exercice 4 : Associer les éléments

Associe chaque organe ou fonction à son rôle :

  • 1. Cœur → ⬜ A. Produit l’air expiré
    2. Poumons → ⬜ B. Pompe le sang
    3. Muscles → ⬜ C. Reçoivent l’oxygène
    4. Vaisseaux sanguins → ⬜ D. Transportent le sang
Correction type :
1 → B ; 2 → A ; 3 → C ; 4 → D
