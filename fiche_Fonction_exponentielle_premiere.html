📖 Définition de la fonction exponentielle

Il existe une unique fonction f, définie et dérivable sur ℝ, vérifiant :

f' = f et f(0) = 1

Cette fonction est la fonction exponentielle, notée exp. On écrit exp(x) = eˣ avec e ≈ 2,718.

💡 Ce que ça signifie (eˣ)' = eˣ — la dérivée est elle-même
e⁰ = 1 — elle vaut 1 en x = 0
📖 Propriétés algébriques — À connaître par cœur
Pour tous réels a et b : eᵃ⁺ᵇ = eᵃ × eᵇ e⁻ˣ = 1 / eˣ eᵃ⁻ᵇ = eᵃ / eᵇ (eᵃ)ⁿ = eⁿᵃ pour tout entier n eˣ > 0 pour tout réel x
Exemples de simplification e²ˣ × e³ˣ = e^(5x)
e^(x+2) / e^(x-1) = e^(x+2−(x−1)) = e³
(e^(2x))³ = e^(6x)
e^(−x) × eˣ = e⁰ = 1
⚠ Erreurs classiques eˣ × eˣ = e^(2x) ≠ e^(x²)
eˣ + eˣ = 2eˣ (pas d'exposant !)
(eˣ)² = e^(2x) ≠ e^(x²)
"On utilise eᵃ⁺ᵇ = eᵃ×eᵇ pour grouper ou séparer. On factorise toujours par la puissance la plus petite."

Q6 Avancé

Montre que eˣ > 0 pour tout réel x en utilisant les propriétés algébriques uniquement.

Pour tout réel x : eˣ = e^(x/2 + x/2) = e^(x/2) × e^(x/2) = (e^(x/2))²
(e^(x/2))² est un carré → (e^(x/2))² ≥ 0
De plus e^(x/2) ≠ 0 (e⁰=1, et par continuité ne s'annule pas)
Donc eˣ = (e^(x/2))² > 0 pour tout réel x. ∎
Q7 Avancé

e^(2x) = (eˣ)² = X². Donc f(x) = X² − 3X + 2
Discriminant : Δ = 9 − 8 = 1. Racines : X₁ = 1, X₂ = 2
Cas X=1 : eˣ = 1 = e⁰ ⟹ x = 0
Cas X=2 : eˣ = 2 ⟹ x = ln(2) ≈ 0,693

📖 Règle fondamentale — Dérivée de eᵘ

Si u est dérivable sur I, alors eᵘ est dérivable sur I et :

(eᵘ)' = u' × eᵘ
Exemples — méthode en 3 étapes (e^(3x+1))' : u=3x+1, u'=3 → 3e^(3x+1)
(e^(x²))' : u=x², u'=2x → 2x·e^(x²)
(e^(−x²+1))' : u=−x²+1, u'=−2x → −2x·e^(−x²+1)
(e^(√x))' : u=√x, u'=1/(2√x) → e^(√x)/(2√x)
💡 Méthode : 3 étapes 1) Identifier u(x) dans l'exposant
2) Calculer u'(x)
3) Écrire : (eᵘ)' = u' × eᵘ
📖 Produit v(x) × eᵘ⁽ˣ⁾

Règle du produit (v·eᵘ)' = v'·eᵘ + v·u'·eᵘ, qu'on factorise par eᵘ :

(v × eᵘ)' = eᵘ × (v' + v·u')
Exemple : g(x) = (1+x)eˣ v = 1+x, v' = 1 · u = x, u' = 1
g'(x) = eˣ(1 + (1+x)·1) = eˣ(2+x)
Exemple : h(x) = 2xe^(−3x²) v = 2x, v' = 2 · u = −3x², u' = −6x
h'(x) = e^(−3x²)(2 + 2x·(−6x)) = 2e^(−3x²)(1−6x²)
"On calcule u' et v', on applique la règle du produit, puis on factorise par eᵘ pour obtenir la forme factorisée."

📖 Injectivité — Résoudre eᵃ = eᵇ

La stricte croissance de exp implique son injectivité :

eᵃ = eᵇ ⟺ a = b eᵃ > eᵇ ⟺ a > b eᵃ < eᵇ ⟺ a < b
Exemples e^(2x+1) = e^(x−3) ⟺ 2x+1 = x−3 ⟺ x = −4
e^(x²) = eˣ ⟺ x²=x ⟺ x(x−1)=0 ⟺ x=0 ou x=1
e^(3x) > e ⟺ 3x > 1 ⟺ x > 1/3
⚠ Si pas de la forme eˣ = e^(cst) Poser X = eˣ (X > 0), résoudre en X, puis eˣ = X₀ ⟹ x = ln(X₀).
"On ramène à eᵃ = eᵇ par injectivité. Si les membres ne sont pas des exponentielles, on pose X = eˣ."
📖 Croissance comparée — À connaître
lim(x→+∞) xⁿ / eˣ = 0 pour tout entier n ≥ 0 lim(x→+∞) eˣ / xⁿ = +∞ lim(x→−∞) |x|ⁿ × eˣ = 0 pour tout entier n ≥ 0
💡 En clair En +∞ : eˣ l'emporte sur tout polynôme (même x^1000).
En −∞ : eˣ "tue" tout polynôme, si raide soit-il.
