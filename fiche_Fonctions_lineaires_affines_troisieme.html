Série 1 — Fonctions linéaires et affines

Définitions, image et antécédent

📖 Fonction linéaire et fonction affine

Une fonction, c est une machine a transformer les nombres : on lui donne un nombre x, elle renvoie un nombre f(x).

Fonction lineaire : f(x) = a x (la droite passe par l origine) Fonction affine : f(x) = a x + b (a = coefficient directeur, b = ordonnee a l origine)
À retenirToute fonction lineaire est une fonction affine particuliere, avec b=0. Si a=0, la fonction est constante : f(x)=b.
"La fonction f est une fonction affine de la forme f(x) = ax + b, avec a le coefficient directeur et b l ordonnee a l origine."
📖 Calculer une image

Calculer l image d un nombre par une fonction, c est remplacer x par ce nombre dans l expression de la fonction et calculer le resultat.

Exemple avec f(x) = 5x + 2 Image de 4 : f(4) = 5x4 + 2 = 20 + 2 = 22 L image de 4 par f est 22
📖 Calculer un antécédent

Calculer l antecedent d un nombre, c est resoudre l equation f(x) = ce nombre.

🔢 Méthode
  • Ecrire l equation f(x) = valeur donnee
  • Remplacer f(x) par son expression ax+b
  • Isoler x en resolvant l equation
Exemple — antécédent de 22 par f(x) = 5x + 2 5x + 2 = 22 5x = 20 x = 4

Série 2 — Déterminer une fonction affine

À partir de deux points connus

📖 Calculer le coefficient directeur

Si on connait deux points A(xA;yA) et B(xB;yB) de la droite representant une fonction affine, on peut calculer le coefficient directeur a.

a = (yB - yA) / (xB - xA) (variation des ordonnees / variation des abscisses)
🔢 Méthode en 3 étapes
  • Calculer a avec la formule des deux points
  • Utiliser un des deux points pour calculer b : remplacer x et y dans y=ax+b
  • Ecrire l expression finale de la fonction f(x)=ax+b
📖 Exemple complet
Déterminer f connaissant A(2;7) et B(5;16) a = (16-7)/(5-2) = 9/3 = 3 On utilise A(2;7) : 7 = 3x2 + b, donc b = 7-6 = 1 f(x) = 3x + 1
0 / 0
⭐⭐ Intermédiaire
Q1 Intermédiaire
Une fonction affine f est telle que sa courbe passe par les points A(2;7) et B(5;16).

Détermine l'expression de f(x), en détaillant le calcul du coefficient directeur et de l'ordonnée à l'origine.

Calcul du coefficient directeur : a = (yB-yA)/(xB-xA) = (16-7)/(5-2) = 9/3 = 3
On utilise le point A(2;7) pour trouver b : 7 = 3x2 + b, soit 7 = 6+b, donc b = 1
L expression de f est : f(x) = 3x + 1

Problèmes — Situations concrètes

Modéliser avec des fonctions affines

Problème 1 — Cartouches d'encre
Avancé

Dans un magasin, une cartouche d encre coute 15€. Sur un site Internet, la meme cartouche coute 10€, avec des frais de livraison fixes de 40€ quel que soit le nombre de cartouches. On note PA(x) le prix a payer au magasin pour x cartouches, et PB(x) le prix a payer sur Internet pour x cartouches. 1) Exprime PA(x) et PB(x). 2) Pour quelle quantite de cartouches les deux prix sont-ils egaux ? 3) A partir de combien de cartouches Internet devient-il plus avantageux ?

1) Au magasin, le prix est proportionnel au nombre de cartouches : PA(x) = 15x (fonction lineaire).
Sur Internet, il y a un prix par cartouche plus des frais fixes : PB(x) = 10x + 40 (fonction affine).
2) On cherche x tel que PA(x) = PB(x) : 15x = 10x + 40, soit 5x = 40, donc x = 8.
Pour 8 cartouches, les deux prix sont egaux : PA(8) = 15x8 = 120€ et PB(8) = 10x8+40 = 120€. Les deux prix sont egaux pour 8 cartouches.
3) Au-dela de 8 cartouches, Internet devient plus avantageux (le prix par cartouche etant plus faible, 10€ contre 15€, l ecart se creuse en faveur d Internet).
