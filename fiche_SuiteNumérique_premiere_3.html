📈 Sens de variation d'une suite numérique

Fiche de révision – Classe de Première · Les 3 méthodes

📌 Définitions à connaître par cœur

Une suite (uₙ) est croissante si pour tout n : un+1 ≥ uₙ
Une suite (uₙ) est décroissante si pour tout n : un+1 ≤ uₙ
Une suite est constante si pour tout n : un+1 = uₙ
Une suite est monotone si elle est entièrement croissante ou entièrement décroissante
Méthode 1 — Signe de un+1 − uₙ

C'est la méthode universelle, applicable dans tous les cas, que la suite soit explicite ou définie par récurrence.

📋 Étapes

1
Calculer un+1 − uₙ en substituant n par n+1 dans la formule de uₙ.
2
Simplifier et factoriser l'expression obtenue.
3
Étudier le signe de cette différence selon les valeurs de n.
4
Conclure : signe ≥ 0 → croissante ; signe ≤ 0 → décroissante.

📐 Exemple — uₙ = n² − 2n

un+1 − uₙ = [(n+1)² − 2(n+1)] − [n² − 2n]
= [n² + 2n + 1 − 2n − 2] − n² + 2n
= n² + 2n − 1 − n² + 2n ← attention aux parenthèses !
= 2n − 1

• Pour n ≥ 1 : 2n − 1 ≥ 1 > 0 → suite croissante à partir de n = 1.
• Pour n = 0 : 2(0) − 1 = −1 < 0 → u₁ < u₀.

Si un+1 − uₙ ≥ 0 → croissante  |  Si un+1 − uₙ ≤ 0 → décroissante
Méthode 2 — Rapport un+1 / uₙ

Cette méthode s'utilise uniquement si tous les termes sont strictement positifs. Elle est particulièrement adaptée aux suites géométriques.

📋 Étapes

1
Vérifier que uₙ > 0 pour tout n.
2
Calculer le rapport un+1 / uₙ.
3
Comparer ce rapport à 1.
4
Conclure selon le tableau ci-dessous.
Si un+1 / uₙ ≥ 1 → croissante  |  Si un+1 / uₙ ≤ 1 → décroissante

📐 Exemple — uₙ = 3ⁿ (pour n ≥ 0)

Tous les termes sont positifs (puissance de 3 > 0). ✔

un+1 / uₙ = 3n+1 / 3ⁿ = 3

3 ≥ 1, donc la suite est croissante.

📐 Exemple 2 — uₙ = (1/2)ⁿ

un+1 / uₙ = (1/2)n+1 / (1/2)ⁿ = 1/2

1/2 ≤ 1, donc la suite est décroissante.

⚠️ Attention ! Si certains termes sont négatifs ou nuls, cette méthode ne fonctionne plus. Utiliser la méthode 1.
Méthode 3 — Fonction associée

Applicable uniquement lorsque la suite est définie par une formule explicite uₙ = f(n). On utilise la dérivée de f pour conclure plus rapidement.

📋 Étapes

1
Identifier la fonction f telle que uₙ = f(n).
2
Calculer la dérivée f'(x).
3
Étudier le signe de f'(x) sur [0 ; +∞[.
4
Conclure sur le sens de variation de la suite.
f croissante sur [0 ; +∞[ → suite croissante  |  f décroissante → suite décroissante

📐 Exemple — uₙ = √(n+1)

On pose f(x) = √(x+1).

f'(x) = 1 / (2√(x+1)) > 0 pour tout x ≥ 0.

f est croissante sur [0 ; +∞[, donc la suite est croissante.

📐 Exemple 2 — uₙ = n² − 4n

On pose f(x) = x² − 4x.

f'(x) = 2x − 4, qui s'annule en x = 2.

• Pour x < 2 : f'(x) < 0 → f décroissante → suite décroissante pour n ≤ 2.
• Pour x > 2 : f'(x) > 0 → f croissante → suite croissante pour n ≥ 2.

⚠️ Limite ! Cette méthode est inutilisable pour les suites définies par récurrence (ex : un+1 = 2uₙ − 3). Dans ce cas, revenir à la méthode 1.

🔍 Tableau comparatif des 3 méthodes

Méthode Formule explicite ? Récurrence ? Termes positifs requis ? Rapidité
1 — Différence ✔ Oui ✔ Oui ✔ Non (tous les cas) Moyenne
2 — Rapport ✔ Oui ✘ Difficile ✘ Oui (obligatoire) Rapide
3 — Fonction associée ✔ Oui ✘ Non ✔ Non Rapide

💡 En cas de doute, la méthode 1 fonctionne toujours. Les méthodes 2 et 3 sont des raccourcis selon le contexte.

✏️ Exercices – Étude complète (à rédiger)

1️⃣ Étudier le sens de variation de uₙ = 3n − 5 (méthode 1).

un+1 − uₙ = [3(n+1) − 5] − [3n − 5] = 3n + 3 − 5 − 3n + 5 = 3 > 0
La différence est strictement positive pour tout n, donc la suite est croissante.

2️⃣ Étudier le sens de variation de uₙ = 5 × (0,8)ⁿ (méthode 2).

Tous les termes sont positifs (5 > 0 et 0,8 > 0). ✔
un+1 / uₙ = 5 × 0,8n+1 / (5 × 0,8ⁿ) = 0,8
0,8 < 1, donc la suite est décroissante.

3️⃣ Étudier le sens de variation de uₙ = √(2n + 3) (méthode 3).

On pose f(x) = √(2x + 3).
f'(x) = 2 / (2√(2x+3)) = 1 / √(2x+3) > 0 pour tout x ≥ 0.
f est strictement croissante sur [0 ; +∞[, donc la suite est croissante.

4️⃣ un+1 = uₙ/3 + 2 avec u₀ = 9. La suite est-elle croissante ou décroissante ?

Suite définie par récurrence → méthode 1.
un+1 − uₙ = uₙ/3 + 2 − uₙ = 2 − 2uₙ/3
On a besoin de connaître le signe de 2 − 2uₙ/3, c'est-à-dire si uₙ < 3 ou uₙ > 3.

Calcul des premiers termes : u₀ = 9, u₁ = 9/3 + 2 = 5, u₂ = 5/3 + 2 ≈ 3,67...
Les termes semblent décroître vers 3. On vérifie : si uₙ > 3, alors 2 − 2uₙ/3 < 0 → suite décroissante (tant que uₙ > 3).

5️⃣ uₙ = n² − 8n + 10. Déterminer le sens de variation selon les valeurs de n.

Méthode 3 : f(x) = x² − 8x + 10.
f'(x) = 2x − 8 = 0 → x = 4.

• Pour n < 4 : f'(n) < 0 → suite décroissante.
• Pour n > 4 : f'(n) > 0 → suite croissante.

La suite est décroissante pour n ≤ 4, puis croissante pour n ≥ 4.
