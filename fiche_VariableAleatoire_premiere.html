📖 I — Déterminer une loi de probabilité

Une variable aléatoire X associe à chaque issue d'une expérience aléatoire une valeur numérique. La loi de probabilité est le tableau qui liste toutes les valeurs possibles et leurs probabilités.

Xx₁x₂…xₙ
P(X=xᵢ)p₁p₂…pₙ
Règle fondamentale : p₁ + p₂ + … + pₙ = 1
Méthode 1) Lister toutes les valeurs possibles de X
2) Calculer chaque probabilité (dénombrement, arbre…)
3) Vérifier que la somme des probabilités vaut bien 1
"X prend les valeurs … avec les probabilités … La somme des probabilités vaut bien 1, donc ce tableau est une loi de probabilité."
🔣 Notations à connaître
XVariable aléatoire
P(X = k)Proba que X vaille k
P(X < k)Proba que X soit < k (strict)
P(X ≤ k)Proba que X soit ≤ k (large)
P(X > k)Proba que X soit > k (strict)
P(X ≥ k)Proba que X soit ≥ k (large)
ΩUnivers (toutes les issues)
📖 XII — Calculer P(X < k) ou P(X > k)

On additionne les probabilités de toutes les valeurs qui vérifient la condition. L'astuce du complémentaire évite les longs calculs :

P(X > k) = 1 − P(X ≤ k) P(X < k) = 1 − P(X ≥ k)
"au moins k"X ≥ k
"au plus k"X ≤ k
"strictement plus"X > k
"strictement moins"X < k
⚠ Piège classique P(X < 3) n'inclut PAS P(X=3) · P(X ≤ 3) l'inclut !
"P(X > 2) = 1 − P(X ≤ 2) = 1 − (p₁ + p₂) = …"

Question 8⭐⭐⭐ Avancé
Un jeu consiste à tirer une boule dans une urne contenant 3 boules rouges, 2 bleues et 1 verte. X = gain : rouge → 0 €, bleue → 2 €, verte → 5 €.

Construis le tableau de loi, puis calcule P(X ≥ 2).

Total : 6 boules
P(X=0) = 3/6 = 1/2 · P(X=2) = 2/6 = 1/3 · P(X=5) = 1/6
Vérification : 1/2 + 1/3 + 1/6 = 3/6 + 2/6 + 1/6 = 6/6 = 1 ✔
P(X ≥ 2) = P(X=2) + P(X=5) = 1/3 + 1/6 = 2/6 + 1/6 = 3/6 = 1/2
Ou : P(X ≥ 2) = 1 − P(X < 2) = 1 − P(X=0) = 1 − 1/2 = 1/2 ✔
Question 9⭐⭐⭐ Avancé
On sait que P(X=1)=0,15 · P(X=2)=0,35 · P(X=3)=p · P(X=4)=0,1 · P(X=5)=q avec p = 2q.

Détermine p et q, puis calcule P(2 < X ≤ 4).

Somme = 1 : 0,15 + 0,35 + p + 0,1 + q = 1 → p + q = 0,4
Avec p = 2q : 2q + q = 0,4 → 3q = 0,4 → q = 2/15 ≈ 0,133 et p = 4/15 ≈ 0,267
P(2 < X ≤ 4) = P(X=3) + P(X=4) = p + 0,1 = 4/15 + 1/10 = 8/30 + 3/30 = 11/30 ≈ 0,367
Attention : X > 2 strict donc X=2 est exclu, X ≤ 4 large donc X=4 est inclus.

📖 II & III — Espérance et interprétation
E(X) = x₁×p₁ + x₂×p₂ + … + xₙ×pₙ
Interprétation E(X) est la valeur moyenne obtenue si on répète l'expérience un très grand nombre de fois.
E(X) > 0Jeu favorable au joueur
E(X) = 0Jeu équitable
E(X) < 0Jeu défavorable au joueur
"E(X) = … donc en moyenne, le joueur [gagne / perd] … euros par partie. Le jeu est [favorable / équitable / défavorable]."
📖 IV & V — Variance et écart-type
V(X) = E(X²) − [E(X)]² E(X²) = x₁²×p₁ + x₂²×p₂ + … + xₙ²×pₙ σ(X) = √V(X)
Interprétation σ(X) mesure la dispersion autour de E(X) dans la même unité que X.
V(X) toujours ≥ 0 · σ(X) toujours ≥ 0
"L'écart-type est σ(X) = … donc les valeurs s'écartent en moyenne de … [unité] par rapport à la moyenne de E(X) = …"
📖 VI · VII · VIII · IX — Linéarité Y = aX + b

Si Y = aX + b alors :

E(Y) = a × E(X) + b V(Y) = a² × V(X)  ←  b disparaît ! σ(Y) = |a| × σ(X)  ←  b disparaît !

Tableau récapitulatif

Transformation
Y = aX + b
Espérance
aE(X) + b
Variance
a²V(X)
Écart-type
|a|σ(X)
⚠ Erreurs classiques · Ne pas mettre a² dans V(Y)
· Inclure b dans V(Y) → b disparaît TOUJOURS !
· Oublier la valeur absolue dans σ(Y) si a < 0
"Y = 3X − 2 donc E(Y) = 3×E(X) − 2 = … · V(Y) = 9×V(X) = … · σ(Y) = 3×σ(X) = …"
📖 XI — Jeu équitable, favorable, défavorable

X représente le gain NET (après déduction de la mise). On cherche k tel que le jeu soit équitable / favorable / défavorable :

Équitable ↔ E(X) = 0 Favorable ↔ E(X) > 0 Défavorable ↔ E(X) < 0
Méthode en 3 étapes 1) Exprimer le gain net X en fonction de k
2) Calculer E(X) en fonction de k
3) Résoudre E(X) = 0 (ou > 0 ou < 0)
"Pour k = …, on a E(X) = 0 donc le jeu est équitable. Pour k < …, E(X) > 0 donc le jeu est favorable au joueur."

Question 8⭐⭐⭐ Avancé
Un jeu coûte k euros. On tire une carte parmi 10 numérotées de 1 à 10. Si le numéro est pair, on gagne 3 €. Si le numéro est un multiple de 5, on gagne 8 €. Sinon on ne gagne rien.

Pour quelle valeur de k le jeu est-il équitable ?

Valeurs paires : 2,4,6,8 → 4 cartes (sauf 10 qui est aussi multiple de 5)
Multiples de 5 : 5,10 → 2 cartes. Attention 10 est pair ET multiple de 5 → on gagne 3€ + 8€ = 11€ ? Non : on prend la règle stricte → carte 10 : multiple de 5 donc 8€
Résumé : cartes 2,4,6,8 → gain 3€ (proba 4/10) · cartes 5,10 → gain 8€ (proba 2/10) · cartes 1,3,7,9 → gain 0€ (proba 4/10)
Gain net X : 3−k (proba 4/10) · 8−k (proba 2/10) · −k (proba 4/10)
E(X) = (3−k)×4/10 + (8−k)×2/10 + (−k)×4/10
= (12−4k+16−2k−4k)/10 = (28−10k)/10
Équitable : E(X)=0 → 28−10k=0 → k = 2,8 €
Question 9⭐⭐⭐ Avancé
X a pour loi : P(X=−2)=1/4 · P(X=0)=1/2 · P(X=4)=1/4. On pose Y = aX + 3 tel que E(Y) = 5.

Détermine a, puis calcule V(Y) et σ(Y).

E(X) = (−2)×1/4 + 0×1/2 + 4×1/4 = −1/2 + 0 + 1 = 1/2
E(Y) = a×E(X) + 3 = 5 → a×(1/2) = 2 → a = 4
E(X²) = 4×1/4 + 0×1/2 + 16×1/4 = 1 + 0 + 4 = 5
V(X) = E(X²) − [E(X)]² = 5 − (1/2)² = 5 − 1/4 = 19/4
V(Y) = a²×V(X) = 16 × 19/4 = 76
σ(Y) = |a|×σ(X) = 4×√(19/4) = 4×√19/2 = 2√19 ≈ 8,72
Question 10 — Problème complet⭐⭐⭐ Avancé
Une entreprise fabrique des pièces. X = nombre de pièces défectueuses sur 100. On sait que E(X) = 5 et σ(X) = 2. Le coût de correction est Y = 3X + 10 euros.

Calcule E(Y) et σ(Y). Interprète E(Y) dans le contexte.

E(Y) = 3×E(X) + 10 = 3×5 + 10 = 15 + 10 = 25 €
V(X) = σ(X)² = 4 → V(Y) = 3²×4 = 36
σ(Y) = |3|×σ(X) = 3×2 = 6 €
Interprétation : En moyenne, le coût de correction est de 25 € par lot de 100 pièces. Ce coût s'écarte typiquement de 6 € autour de cette moyenne.
