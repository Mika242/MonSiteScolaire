Écrire en fonction de x

Traduire une phrase en expression algébrique.

📖 Cours

Une lettre (souvent x) représente un nombre quelconque ou inconnu. On traduit une phrase mathématique en utilisant les opérations :

« augmenté de 5 » → x + 5
« le double » → 2x
« diminué de 7 » → x − 7
« le tiers, augmenté de 4 » → x/3 + 4
« le carré » → x²
Exemple Périmètre d'un rectangle de longueur x et de largeur 3 :
P = 2 × (x + 3) = 2x + 6

Réduire une somme algébrique

Regrouper les termes semblables.

📖 Cours

Termes semblables = même lettre, même degré. On les additionne ou soustrait.

3x + 5x = (3+5)x = 8x
7x − 2x + 4 = 5x + 4
4x² + 2x − x² + 5x = 3x² + 7x
Attention ! On ne peut pas additionner x et x², ni x et un nombre seul.
2x + 3 ≠ 5x

Substituer une lettre par une valeur

Remplacer x par un nombre et calculer.

📖 Cours

Substituer = remplacer la lettre par un nombre donné, puis calculer.

Si x = 4, alors 3x + 2 = 3×4 + 2 = 14
Si x = 3, alors x² − 1 = 3² − 1 = 8
Conseil Toujours mettre la valeur entre parenthèses pour éviter les erreurs :
x = −2 → 3x = 3×(−2) = −6

⭐⭐⭐ Avancé

Vérifie si les expressions A = x² + x et B = x(x+1) donnent le même résultat pour x = 6 et pour x = −4.

Pour x = 6 : A = 36 + 6 = 42 et B = 6×7 = 42 ✔
Pour x = −4 : A = 16 + (−4) = 12 et B = (−4)×(−3) = 12 ✔
Conclusion : A et B sont égales dans les deux cas. C'est normal car x(x+1) = x² + x (développement).

Vérifier si un nombre est solution d'une équation

Tester une valeur dans une équation.

📖 Cours

Une équation est une égalité contenant une inconnue. Vérifier une solution = substituer et tester l'égalité.

2x + 1 = 7 → tester x = 3 : 2(3)+1 = 7 ✔ → solution !
2x + 1 = 7 → tester x = 2 : 2(2)+1 = 5 ≠ 7 ✘ → pas solution
Méthode 1) Remplace x par la valeur testée
2) Calcule le membre gauche et le membre droit séparément
3) Si les deux membres sont égaux → c'est une solution

Question 7
⭐⭐⭐ Avancé

Un rectangle a un périmètre de 26 cm. Sa largeur est x et sa longueur est (x + 3). Quelle équation traduit cette situation ? Puis vérifie que x = 5 est la solution.

Périmètre = 2 × (longueur + largeur)
Équation : 2(x + x + 3) = 26, soit 2(2x + 3) = 26
Vérification avec x = 5 :
2(2×5 + 3) = 2×13 = 26 ✔
Oui, x = 5 est solution ! La largeur est 5 cm et la longueur est 8 cm.
Question 8 — Problème complet
⭐⭐⭐ Avancé

Marie pense à un nombre. Elle le multiplie par 3, lui ajoute 5 et obtient 20. Traduis cela par une équation, puis vérifie que x = 5 est la solution.

Traduction : 3x + 5 = 20
Vérification avec x = 5 :
Membre gauche : 3×5 + 5 = 15 + 5 = 20
Membre droit : 20
20 = 20 ✔ → x = 5 est bien la solution.
Marie avait pensé au nombre 5.
