Suites numériques – Première

1) Notion de suite numérique – Rappels de cours

  • Une suite associe à chaque entier n un nombre noté uₙ
  • Le nombre uₙ est appelé terme de rang n
  • Une suite peut être définie :
    • explicitement : uₙ = f(n)
    • par récurrence : u₀ donné et uₙ₊₁ = f(uₙ)
Exemple : uₙ = 2n + 1 ⇒ u₃ = 7

2) Suites arithmétiques – Rappels de cours

  • uₙ₊₁ = uₙ + r
  • uₙ = u₀ + n × r
  • r est la raison (différence constante)
Exemple : u₀ = 2, r = 3 ⇒ u₄ = 14

Exercices

1️⃣ uₙ = 5 + 2n est :

3) Suites géométriques – Rappels de cours

  • uₙ₊₁ = q × uₙ
  • uₙ = u₀ × qⁿ
  • q est la raison
Exemple : u₀ = 3, q = 2 ⇒ u₄ = 48

  • Une valeur initiale est indispensable
  • Chaque terme dépend du précédent
Exemple : u₀ = 1, uₙ₊₁ = 2uₙ
